安装MySql-阿里云linux服务器

发表于 2018-07-06

打开下面的地址，复制对应版本.rpm文件的地址

https://dev.mysql.com/downloads/repo/yum/

通过wget下载.rpm文件

将.rpm文件添加到yum仓库

sudo rpm -Uvh mysql57-community-release-el6-n.noarch.rpm

查看状态

yum repolist all | grep mysql

安装mysql

sudo yum install mysql-community-server

查看数据库状态

sudo service mysqld status

开启数据库

sudo service mysqld start 或 sudo systemctl start mysqld.service

关闭数据库

sudo service mysqld stop

查看安装后生成的默认密码

sudo grep 'temporary password' /var/log/mysqld.log

通过查询到的密码，在服务器上使用root用户登录，并更改密码

mysql -uroot -p

ALTER USER 'root'@'localhost' IDENTIFIED BY 'MyNewPass4!';

因为root用户默认是禁止远程连接的，创建一个新的用户

创建用户admin并设置密码

CREATE USER 'admin'@'%' IDENTIFIED BY 'some_pass';

给这个用户授予所有的远程访问的权限。这个用户主要用于管理整个数据库、备份、还原等操作。

GRANT ALL ON *.* TO 'admin'@'%';

使授权立即生效

flush privileges;

退出mysql环境

exit

设置防火墙，将端口号开启（注：如果是阿里云服务器开启端口需要在安全组里设置）

firewall-cmd --permanent --zone=public --add-port=3306/tcp

firewall-cmd --permanent --zone=public --add-port=3306/udp

firewall-cmd --reload
